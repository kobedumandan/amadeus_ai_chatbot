## amadeus AI Chatbot 🗨️🤖

This is amadeus, an AI chatbot wrapper using OpenRouter API (free).

It does everything a normal AI chatbot can, utilizing free models to give a free experience like other LLM chatbots like ChatGPT and Claude, but with higher usage.
